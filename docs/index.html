Namegen

Fast, deterministic name generation across dozens of languages

Get Started

Features

Multiple Profiles

Generate realistic names in English, Japanese, Spanish, Nordic, Slavic, Tamil and many other languages.

Deterministic Randomness

Pass a seed with -s to produce reproducible results - perfect for games, testing and world‑building.

Realism Control

Tune the realism of generated names from 0 (fictional phonotactics) to 100 (curated and real‑looking) with -realism.

Gender Hints

Specify male, female or neutral hints for more culturally appropriate names.

Flexible Output

Toggle surnames (-l), reverse order (-r) or generate batches of names (-c) with ease.

Standalone Binary

No CGO or external plugins - everything is compiled into a single static binary for easy distribution.

Getting Started

Namegen is a Go application. Build the CLI and run it with your chosen flags:

$ make build
$ ./bin/namegen -mode japanese -l -realism 80 -s 42 -gender female

The command above generates ten female Japanese names with high realism using seed 42. Namegen supports many other modes and flags - see below for a complete reference.

CLI Flags

When buiilt into a binary, namegen accepts flags in the command line as previously mentioned:

Flag Meaning
-mode <name> Profile/mode name (compiled-in). Defaults to English.
-l Include last name
-r Reverse output order (last first)
-gender <male, female, neutral> Gender hint passed to profile
-family <key> Optional “family override” (profiles may interpret it differently)
-realism 0...100 0 = fictional phonotactics, 100 = curated / real-looking
-s <seed> Seed (0 or omitted = random each run)
-c <count> Number of names to generate
-d Dev mode: prints config JSON
-p List all available profiles

How It Works

Namegen implements each language as a "profile" that knows how to combine phonemes and syllables in a realistic way. The CLI passes an api.ProfileConfig to the selected profile. Profiles call api.NewRand(cfg) which returns a deterministic random number generator when a seed is provided and a time‑seeded RNG when it isn't. Realism is controlled by the profile itself, switching between fictional phonotactics and curated lists. For full reproducibility, always pass a non‑zero seed.

Contributing

Contributions are welcome! If you'd like to add a new profile, create a new directory under plugins/ and register it via api.RegisterProfile. See the README for a minimal profile skeleton and development instructions. Please ensure that make ci passes before opening a pull request.
